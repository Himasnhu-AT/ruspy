# Ruspy syntax

## 1. declare variables:

- type defined at execution

```
a = 10
```

a with value 10

- type declared

```
a: int = 10
```

a of type int is 10

### Allowed variables:

- int: integer
- int32
- int64
- float
- float32
- float64
- char
- str
- str8
- str32
- str64

## 2. declaring function:

```
def func_name(args) {
    // function code (execution)
}
```
